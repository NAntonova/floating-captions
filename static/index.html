💡 FaceTime Audio Capturing Guide:

  1. Select Mixed or System Audio mode and click Start Listening.
  2. Chrome will prompt you to share a screen or window. Choose the FaceTime window or your Entire Screen.
  3. CRITICAL: Check the "Share system audio" checkbox in the bottom-left corner of the Chrome share popup! Otherwise FaceTime audio won't be captured.
  4. Safari/Firefox Fallback: Install BlackHole (`brew install blackhole-2ch`), create an Aggregate Device with your Mic, set FaceTime output to a Multi-Output Device, and select it as your browser Mic source.
